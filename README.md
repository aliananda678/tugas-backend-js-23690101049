# Tugas Mandiri Dasar Back-End Development

## Deskripsi Aplikasi
Aplikasi web sederhana ini dibuat untuk memahami konsep Server-Side Scripting dan HTTP Request.  
Aplikasi memisahkan:
- Front-End: Formulir HTML (index.html)
- Back-End: Server Node.js (server.js)

## Fitur
Formulir berisi input:
- Nama  
- Email  
- Pesan (message)  
- Nomor HP

Data yang dikirim dari form diproses oleh server menggunakan metode HTTP POST.

## Cara Menjalankan

1. Pastikan Node.js sudah terinstal
2. Buka terminal pada folder proyek:

   npm install  
   node server.js

3. Buka file index.html melalui browser
4. Isi formulir → klik kirim
5. Server akan menerima request dan menampilkan data di terminal.

## Tujuan Pembelajaran
Mahasiswa dapat:
- Memahami konsep request–response HTTP  
- Memahami peran back-end dalam memproses data  
- Menerapkan pemisahan logika front-end dan back-end.
